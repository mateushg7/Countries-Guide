# 🌍 Countries Guide

Aplicação web simples que permite pesquisar e visualizar informações sobre países do mundo, utilizando a REST Countries API.

## 🔧 Tecnologias
- HTML
- CSS
- JavaScript (Vanilla)
- Fetch API

## 🚀 Funcionalidades
- Busca por país pelo nome
- Exibição de bandeira, capital, população e região
- Layout responsivo
